## 如何使用

### 1. 构筑后端项目

```bash
cd backend
conda create -n ai-fantasy-assistant python=3.11
conda activate ai-fantasy-assistant
pip install -r requirements.txt
```

### 2. 运行后端
```bash
python run.py
```

### 3. 构筑前端项目

```bash
cd frontend
npm install
npm run build
```

### 4. 运行前端
```bash
npm start
```
